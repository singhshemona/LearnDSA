Knuth-Morris-Pratt (KMP)

Description

KMP is a process to compare a pattern to a given text from left-to-right of the pattern, shifting the pattern more intelligently than brute-force.

Procedure

  1. Build the prefix table (aka failure function) which is the table showing the size of the largest prefix of P that is also a suffix of P.
  2. Lining up the first letter of the pattern with the first letter of the text, compare pattern to text, starting from leftmost character in the pattern.
  3. If mismatch, check the table for the value at the P[whatever index you are on] and skip forward that many number of spaces.

Examples

Matching up a specific DNA sequence to a larger one, similarities between the two strings can show similarities in gene formation.
