Heap Sort O(nlogn)

Description

Traditional sorting algorithm for a given binary tree, an array, or a heap, where we turn data into a heap (if not already), use the heap to sort the array/tree by removing the max number and putting it at the end of the array/tree, continue to do step 2 until the heap is empty and the array/tree is now completely sorted.

Procedure

Runtime Explaination

the While loop is done n times since we remove one of the nodes each time, so time for while loop is O(n log n). Total time is therefore O(n log n) + O(n log n), or just O(n log n)
