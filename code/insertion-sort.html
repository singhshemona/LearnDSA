Insertion Sort O(n2)

Description

Insertion Sort is an excellent algorithm for sorting small sequences (less than 50 elements), since it is simple to program and small sequences necessarily have few pairs out of order. It is also effective if the sequence is almost sorted.

Procedure

The algorithm proceeds by considering one element at a time, placing the element in the correct order relative to those before it. We start with the first element in the array, which is trivially sorted by itself. When considering the next element in the array, if it is smaller than the first, we swap them. We then move onto the third element and compare it with the elements before it, swapping as needed to put it in the right place. We continue this process until the array is sorted.

Runtime Explaination

In the worst case possible every element needed to be sorted is less than every element to its left. This would mean the array starts out in reverse order, making it run in O(n2).

You insert the new card in the right place, and once again, your hand holds fully sorted cards. You repeat the procedure with any more new cards you are given. You only do this one at a time.
